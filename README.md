# Minecraft2D

A 2D version of Minecraft built with PyGame.

## Features
- Simple 2D grid-based world
- Player movement
- Basic block placement and destruction

## Requirements
- Python 3.8+
- PyGame

## Installation
1. Clone this repository
2. Install requirements:
   ```
   pip install -r requirements.txt
   ```

## Running the Game
```bash
python main.py
```
